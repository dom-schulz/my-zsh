# my-zsh

A versioned home for my Zsh configuration. This repository exists so I can:

- **Store** my `.zshrc` and any related shell config in one place  
- **Track** changes over time and keep a history of tweaks  
- **Sync** the same setup across machines by cloning and linking (or copying) into `~`

No tooling or automation—just the config files themselves. Add your `.zshrc` (and optional split files) when ready; usage and layout can be documented here once they’re in place.
